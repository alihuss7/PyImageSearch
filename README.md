# PyImageSearch University

A comprehensive collection of computer vision and deep learning projects covering OpenCV fundamentals, advanced image processing techniques, and TensorFlow object detection.

## 📚 Course Structure

This repository is organized into four main learning tracks:

### 🎨 [OpenCV 101 - Fundamentals](OpenCV101/)
Introduction to basic OpenCV operations and image manipulation techniques.

**Topics Covered:**
- Loading and displaying images
- Image arithmetic and bitwise operations
- Color space conversions
- Geometric transformations (resizing, rotating, flipping, translating)
- Cropping and masking
- Channel splitting and merging
- Drawing shapes on images

### 🔍 [OpenCV 102 - Intermediate Techniques](OpenCV102/)
Advanced image processing including edge detection, filtering, and morphological operations.

**Topics Covered:**
- Smoothing and blurring (Gaussian, median, bilateral filtering)
- Thresholding (simple, Otsu's, adaptive)
- Edge detection (Canny, Sobel, Scharr)
- Image gradients and morphological operations
- Convolutions and kernels
- Auto-tuning edge detection parameters

### 📊 [OpenCV 103 - Advanced Processing](OpenCV103/)
Professional-level image enhancement and analysis techniques.

**Topics Covered:**
- Histogram equalization and analysis
- Gamma correction
- Color correction and matching
- Low-contrast image detection
- Advanced color manipulation

### 🤖 [DeepLearning 101](DeepLearning101/)
Introduction to machine learning and deep learning for computer vision.

**Topics Covered:**
- K-Nearest Neighbors (k-NN) classifier
- Custom dataset loading and preprocessing
- Building your first image classifier
- Feature extraction and representation

### 🎯 [TensorFlow Object Detection Course](TFODCourse/)
Complete end-to-end object detection pipeline using TensorFlow.

**Topics Covered:**
- Custom image collection and labeling
- TensorFlow Object Detection API setup
- Training custom object detection models
- Real-time object detection
- Model evaluation and optimization

## 🚀 Getting Started

### Prerequisites
```bash
# Python 3.7 or higher
python --version

# Required libraries
pip install opencv-python numpy matplotlib scikit-learn imutils
```

### Installation
```bash
# Clone this repository
git clone https://github.com/YOUR_USERNAME/PyImageSearchUniversity.git
cd PyImageSearchUniversity

# Install dependencies
pip install -r requirements.txt
```

### Project Navigation

Each course folder contains:
- 📓 **Jupyter Notebooks** - Interactive tutorials with code and explanations
- 🐍 **Python Scripts** - Standalone scripts that can be run from the command line
- 📁 **Dataset/Images** - Sample images and datasets for practice

## 📖 How to Use This Repository

1. **Start with OpenCV 101** if you're new to computer vision
2. **Progress sequentially** through OpenCV 102 and 103 for advanced techniques
3. **Explore DeepLearning 101** to understand machine learning fundamentals
4. **Build with TFODCourse** to create a complete object detection project

Each module is self-contained and can be studied independently based on your interests.

## 🛠️ Common Commands

### Running Python Scripts
