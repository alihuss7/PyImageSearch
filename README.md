# PyImageSearch University

Structured computer vision and deep learning practice projects, organized by topic and difficulty.

## Repository Structure

- [OpenCV101](OpenCV101/): Core OpenCV operations and image manipulation basics.
- [OpenCV102](OpenCV102/): Intermediate processing techniques including thresholding, edges, gradients, and morphology.
- [OpenCV103](OpenCV103/): Advanced image enhancement and analysis workflows.
- [DeepLearning101](DeepLearning101/): Introductory machine learning and image classification exercises.
- [TFODCourse](TFODCourse/): TensorFlow Object Detection API training and inference pipeline.

## Prerequisites

- Python 3.7+

## Setup

```bash
pip install -r requirements.txt
```

## Usage

Run Python scripts:

```bash
cd OpenCV101/opencv-load-image
python load_image_opencv.py
```

Run notebooks:

```bash
jupyter notebook
```

## Content Format

- Jupyter notebooks for guided walkthroughs
- Python scripts for direct execution
- Local datasets/images where required by each module
